# Controlling and accessing the interface

### Controlled by the Server/Client Model

- Orders window
- Balances window
- Annotations on the candlestick chart
- News window
- User defined windows

### Controlled by plugins

- Orderbook window
    - Depth chart
- Symbols window (Exchange Window)
- Candlestick charts
